Built-In Elements Collection

About

This is a Vanilla JavaScript `Custom Elements Collection`. They are Non-Autonomous - or `Built-in Custom Elements` - focused on behavior with a minimalistic approach. Each element has a single purpose, extends a Standard HTML Element and has a basic API.

The elements on this page has a very minimal style and a rich and complex behavior. The intention here is encapsulate a rich set of complex behavior elements and let you free to style them as your want/need. The page you are seeing is almost completely made with built-in elements.

This page makes use of the elements collection to provide a set of examples and demonstrate its powers. The use is very simple: just an HTML tag with a `is` attribute telling how the tag will be extended. Built-in custom elements are a specification part of the Web Components official spec. You can check the github documentation of this collection for further information.

Usage

You must load the proper JS and CSS files on the page. Add the polyfill which is needed by safari, and the dependencies in case you element needs one (only the el-mustache has a dependency).

You use the element as a normal HTML Element, but specify the name of the custom element inside the `is` attribute. You must see the documentation to know which tag to use as which `is` value to put on your tag.

See the documentation on github for further information. You also use this page as a guide

Benefits

  • Vanilla JS. No dependencies. No framework.
  • Add all elements, or only those that you will need.
  • Behavior driven, you are free to add your styles.
  • No Shadow-DOM
  • Inherit from Standards, inheriting all the common elements structure
  • Semantic HTML with no strange custom tag names
  • SEO Friendly and Accessibility friendly
  • Reusable and Minimal
  • Provides separation of concerns between data, behavior and style.

Styling

This is a standard HTML elements extended, so you can freely style it, apply your own classes, ids, attributes, etc. You can manage/change the Lightbox size, position, z-index and everything else by CSS. You may use the [is="{element-name}"] element selector to target the element on your CSS.
